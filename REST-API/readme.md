# REST-API

This project consists of user management system which enables user to have CRUD operations on top of User Module

## Introduction

Project manages users with the ability to apply CRUD operations on Users. Application is builded with Gin Framework (Go programming language) for faster shipping and easy to be scaled out. For Database needs application is utilizing PostgresSQL.


## Feautures

 - extensible design 
 - performance oriented
 - fail proof with graceful shutdown
